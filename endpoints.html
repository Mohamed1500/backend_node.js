API Documentation

Welkom op de api documentatie. Hier kan je de details zien over de endpoints en hoe je ze moet gebruiken.

Endpoints

GET /api/users

Haal een lijst van users met optionele filters en pagination.

  • page (integer, optioneel): Aantal page voor pagination (default: 1).
  • limit (integer, optioneel): Aantal users per page (default: 10).
  • name (string, optioneel): Filter users per name.
  • age (integer, optioneel): Filter users per age.
Vb: GET http://localhost:3000/api/users?page=2&limit=5

POST /api/users

Maak een nieuw user door de volgende informatie in de request body te sturen:

  • firstName (string, vereist):De user's voornaam.
  • name (string, vereist): De user's achternaam.
  • age (integer, vereist): De user's leeftijd.
  • email (string, vereist): De user's email.
  • password (string, vereist): De user's wachtwoord.
Vb: POST http://localhost:3000/api/users

PUT /api/users/:id

Update een bestaande user per ID. Stuur de updated informatie in de request body:

  • firstName (string, optioneel): De user's voornaam.
  • name (string, optioneel): De user's achternaam.
  • age (integer, optioneel): De user's leeftijd.
  • email (string, optioneel): De user's email.
  • password (string, optioneel): De user's wachtwoord.
Vb: PUT http://localhost:3000/api/users/1

DELETE /api/users/:id

Delete a user by ID.

Vb: DELETE http://localhost:3000/api/users/1
